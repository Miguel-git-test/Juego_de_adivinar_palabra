# 🏰 Wordle de Íscar

Un juego de adivinar palabras tematizado sobre la Villa de Íscar. Explora la cultura, los monumentos y las tradiciones de Íscar mientras pones a prueba tu ingenio.

## 🚀 Cómo publicar en GitHub Pages

Para que el juego sea accesible desde una URL pública (ej. `https://tu-usuario.github.io/juego-iscar/`), sigue estos pasos con **GitHub Desktop**:

1. **Crear el repositorio**:
   - Abre GitHub Desktop.
   - Ve a `File` -> `Add Local Repository`.
   - Selecciona la carpeta donde están estos archivos.
   - Si no es un repositorio aún, te pedirá "Create a repository here". Hazlo.
   
2. **Subir a GitHub**:
   - Pulsa el botón `Publish repository`.
   - Asegúrate de que el nombre sea el que quieres (por ejemplo `iscar-wordle`).
   - Mantén la opción "Keep this code private" **desmarcada** para poder usar GitHub Pages gratis.

3. **Activar GitHub Pages**:
   - Ve a tu cuenta de GitHub en el navegador.
   - Entra en el repositorio que acabas de subir.
   - Ve a la pestaña **Settings** (Configuración).
   - En el menú de la izquierda, busca **Pages**.
   - En "Build and deployment", asegúrate de que esté seleccionado "Deploy from a branch".
   - En "Branch", selecciona `main` (o `master`) y la carpeta `/ (root)`.
   - Pulsa **Save**.

4. **¡Listo!**: En unos minutos, GitHub te dará una URL (ej. `https://migue.github.io/iscar-wordle/`) donde podrás jugar y compartir el juego.

## 📱 Instalación en Móvil (PWA)
Este juego es una PWA (Progressive Web App). Una vez lo abras en el navegador de tu móvil:
- **iOS (Safari)**: Pulsa el botón "Compartir" y selecciona "Añadir a la pantalla de inicio".
- **Android (Chrome)**: Pulsa los tres puntos y selecciona "Instalar aplicación".
